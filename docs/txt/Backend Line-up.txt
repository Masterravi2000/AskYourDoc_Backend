Step 1: Set up Python Environment

Create a virtual environment:

cd AskYourDoc_Backend
python -m venv venv

Activate it:

Windows: venv\Scripts\activate

Mac/Linux: source venv/bin/activate

Upgrade pip:

pip install --upgrade pip




Step 2: Install Core Dependencies

You’ll need packages for FastAPI, embeddings, file handling, and FAISS:

pip install fastapi uvicorn sentence-transformers faiss-cpu python-multipart PyMuPDF pdfplumber pytesseract pandas python-pptx openpyxl

Optional for images: pip install pillow




Step 3: Basic FastAPI Structure

Create a folder structure:

AskYourDoc_Backend/
├─ app/
│  ├─ main.py
│  ├─ models.py   # for request/response schemas
│  ├─ utils.py    # text extraction, chunking
│  ├─ vectorstore.py  # FAISS setup + embeddings
├─ docs/          # folder to save uploaded files
├─ README.md

In main.py start a minimal FastAPI app:

from fastapi import FastAPI, UploadFile, File
app = FastAPI()

@app.get("/")
def home():
    return {"message": "AskYourDoc Backend is live!"}

@app.post("/upload")
async def upload_file(file: UploadFile = File(...)):
    file_location = f"docs/{file.filename}"
    with open(file_location, "wb") as f:
        f.write(await file.read())
    return {"filename": file.filename, "status": "uploaded"}

✅ Now you can test upload via Postman.




Step 4: File Type Processing

In utils.py write functions to extract text from:

PDF → PyMuPDF / pdfplumber

Images → pytesseract

Excel → pandas.read_excel()

PowerPoint → python-pptx

TXT → plain read

Goal: return plain text from any document.




Step 5: Chunk Text

Split text into manageable chunks (500–1000 words) for embeddings.




Step 6: Generate Embeddings

Use sentence-transformers:

from sentence_transformers import SentenceTransformer
model = SentenceTransformer('all-MiniLM-L6-v2')
vector = model.encode("Your text chunk here")

Each chunk → embedding vector




Step 7: Store in FAISS

Create vectorstore.py:

import faiss
import numpy as np

dimension = 384  # depends on embedding model
index = faiss.IndexFlatL2(dimension)  # simple L2 index
# add embeddings
index.add(np.array([vector], dtype='float32'))

Store metadata (filename, page, chunk) in a parallel list/dictionary.




Step 8: Query Handling

Convert user query → embedding → FAISS → fetch top relevant chunks

Return chunks to local LLM (like Ollama) for final answer generation
